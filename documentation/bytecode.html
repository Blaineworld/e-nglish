Engle Docs: Bytecode

This file documents the format of Engle's bytecode files and the instructions they use. Unlike every actual instruction set ever, Engle bytecode is designed to mimic the structures found in a traditional high-level programming language, but in a more low-level form. (Engle takes a long time to interpret so it runs faster if it's compiled-ish first.) It can also be transcompiled fairly easily, making it convenient as heck.

Instructions

Hex Code Description Operands
00 No Operation
01 Jump By int64: Addresses
07 Begin Block uint8: Layers
08 Terminate
09 Jump To uint64: Address
0F End Block uint8: Layers
7F Evaluate expression
FF Log expression: Message
