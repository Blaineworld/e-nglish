Engle Docs: Inline Text Operations

This file documents Engle's default inline operations for text values, meaning combinations of words that can be used within an expression to manipulate text.

Changing Case

<text> in alternating case
changes letters to alternating case wherever possible.
soME TExt.more Text → sOmE TeXt.mOrE TeXt
<text> in inverse case
changes letters to inverse case wherever possible.
soME TExt.more Text → SOme teXT.MORE tEXT
<text> in lower case
changes letters to lower case wherever possible.
soME TExt.more Text → some text.more text
<text> in normal case
does not change the case of the letters.
soME TExt.more Text → soME TExt.more Text
<text> in proper case
changes letters to proper case wherever possible.
soME TExt.more Text → Some Text.More Text
<text> in sentence case
changes letters to sentence case wherever possible.
soME TExt.more Text → Some text.More text
<text> in upper case
changes letters to upper case wherever possible.
soME TExt.more Text → SOME TEXT.MORE TEXT

Reordering

<text> backwards
reverses the text.
soME TExt.more Text → txeT erom.txET EMos
<text> forwards
does not reverse the text.
soME TExt.more Text → soME TExt.more Text
