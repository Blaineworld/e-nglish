Engle Docs: Inline Numeric Operations

This file documents Engle's default inline operations for numeric values, meaning combinations of words that can be used within an expression to manipulate numbers.

Arithmetic

<number> divided by <number>
performs division.
<number> minus <number>
performs subtraction.
<number> mod <number>
performs modulo.
<number> plus <number>
performs addition.
<number> times <number>
performs multiplication.
<number> to the power of <number>
performs exponentiation.
